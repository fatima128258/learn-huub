# 📚 Learn Hub (Next.js Full-Stack LMS)

A **complete Learning Management System built using **Next.js (frontend + backend)**. This platform supports **Students, Instructors, and Admin roles**, with full workflow tracking from course creation to purchase and engagement.

---

## 🚀 Core Features

### 👩‍🎓 Student

* Signup / Login Authentication
* Browse & Buy Courses
* Watch Instructor Playlists
* Comment & Interact on Content
* Track Learning Progress (step-by-step)
* Dashboard with activity tracking
* Real-time chat system

### 👨‍🏫 Instructor

* Create Courses & Playlists
* Upload Lessons (structured sequence)
* Manage Content Flow
* Track student engagement

### 🛡️ Admin

* Approve Instructor Content (Playlists/Courses)
* Manage Users (Students & Instructors)
* Monitor Platform Activity

---

## 💰 Additional Features

* Balance / Purchase System (course buying)
* Activity Tracking (user progress monitoring)
* Role-based Access Control
* Structured Learning Flow (sequence-based)

## ⚙️ Setup Instructions

### 1. Clone Repository

```
git clone https://github.com/your-username/learn-hub.git
cd learn-hub
```

### 2. Install Dependencies

```
npm install
```

## ▶️ Run Project

```
npm run dev
```

App will run on:
👉 [http://localhost:3000](http://localhost:3000)
